1. pastikan kalian memiliki pembuat server lokal seperti xampp atau laragon

2. pastikan server mySQL menyala tambahkan apache juga agar melihat database menggunakan GUI seperti phpmyadmin atau semacamnya

3. pastikan kalian memiliki database yang products dan categories

4. buka folder project dan jalankan pada terminal "node server.js"

5. jika halaman di cek memunculkan pesan error sebagai judul dan tulisan "cannot GET /" maka aman

6. gunakan postman atau insomnia sebagai aplikasi percobaan

7. pada pencobaan kalian bisa gunakan metode seperti GET, POST, DELETE dan lain-lain tergantung dari ketentuan yang diberikan routes
